# Routini: Gym Video Planner

Smarter workouts, less scrolling. Organize YouTube workouts into clean, quick-access playlists. Built for the gym.

## Getting Started

1. `npm install`
2. Add your `.env.local` with `MAILERLITE_API_KEY`
3. Run `npm run dev`
4. Deploy to Vercel!

## Stack

- Next.js (App Router)
- Tailwind CSS
- MailerLite API
- shadcn/ui
